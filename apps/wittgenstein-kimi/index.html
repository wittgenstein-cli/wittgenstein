thesis

Agent = Model + Harness — extended with modality codecs. The LLM stays the planner; the harness owns routing, schemas, validation, retries, budgets, and traces.

why it scales

Industry harnesses handle tools, memory, and control flow. Wittgenstein adds a clear seam: natural structured IR plus deterministic renderers and optional tiny adapters. Post-training can move to the harness bundle instead of the base model when that trade-off wins. Runs leave artifacts under artifacts/runs/.

02

layers

Five explicit layers map to packages — so contributors cannot implement the idea in the wrong place.

L1

Harness / runtime — router, retry, seed, budget, telemetry, sandbox.

L2

IR / codec — zod schemas, prompt contracts, parse only structured success paths.

L3

Renderer / decoder — IR to bytes; frozen decoders in-bounds, generators out-of-scope.

L4

Optional adapter — small learned translators (e.g. scene → latents) shipped beside codecs.

L5

Packaging — CLI, install scripts, skills, docs as system of record.

03

pipeline

Sole image path (locked): semantic JSON → adapter → frozen decoder → PNG. Decoder ≠ generator; every run writes a trace.

# Docs Archive

This folder is reserved for **historical docs surfaces that no longer belong on an active
decision path**, but are still worth keeping for provenance.

Use this folder sparingly.

Preferred rule:

- archive **near the original surface first** when that keeps history clearer
- use `docs/archive/` only when the material no longer cleanly belongs to one live docs
  surface

Examples that should usually stay surface-local:

- old exec plans -> `docs/exec-plans/archive/`
- research captures -> `docs/research/...`

Move material here when:

- it is genuinely historical
- it has value beyond git history alone
- it would confuse contributors if left in the active tree
- there is no better surface-local archive home

Every archived file or subfolder here should say:

1. why it was archived
2. what superseded it, if anything
3. whether it is still safe to cite, or only safe to read as history

Do not treat this folder as current doctrine, current research scope, or live execution
guidance.
